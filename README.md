# 브런치 글 수정

사용자 글을 로컬에서 교정하고, 글 기반 이미지 프롬프트를 제안하는 웹서비스입니다.

## 현재 동작 방식
- 교정: 브라우저 내 로컬 교정 규칙으로만 수행
- 이미지: 생성하지 않고 프롬프트만 제안(ChatGPT/Canva/Gemini/Grok)

## 주요 기능
1. 오타/띄어쓰기/문장부호 로컬 교정
2. 원문 최소 수정 원칙 반영
3. 변경분 `**Bold**` 표시
4. 변경 이력 + 수정 이유 자동 기록
5. 4인 전문가 토론 형식의 편집자 합의 의견 출력
6. 이미지 프롬프트 자동 제안(ChatGPT/Canva/Gemini/Grok)
7. 교정 결과 전체 복사
8. 일일 명언(24시간 주기 자동 변경)

## 실행 방법
1. `index.html`을 브라우저에서 열기
2. 원문 입력 후 `교정 실행`
3. 이미지 섹션에서 `이미지 프롬프트 제안`

## 참고
- 온라인 교정 사이트/API 연동은 기술적으로 가능하지만, 현재 버전은 요청에 맞춰 로컬 교정으로 고정되어 있습니다.
